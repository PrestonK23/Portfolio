## Featured Projects 
### Dodgers Attendance Analysis
Analyzed attendance trends for Los Angeles Dodgers home games using R and statistical visualization techniques. The project explored how promotions, weekdays versus weekends, and weather conditions impacted attendance patterns. Key findings showed promotional events significantly increased weekday attendance.

## Childcare Affordability Analysis
Created a multi-medium data storytelling project analyzing childcare affordability across the United States using dashboard-style visuals, time-series analysis, and infographic reporting.
